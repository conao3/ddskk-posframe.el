* Overview

ddskk-posframe provides a modern tooltip display for [[https://github.com/skk-dev/ddskk][ddskk]] using [[https://github.com/tumashu/posframe][posframe]].

This package displays SKK conversion candidates in a floating frame, offering a cleaner and more visually appealing experience compared to the default tooltip.

** Related Packages

ddskk-posframe is part of the posframe ecosystem:

- [[https://github.com/tumashu/posframe][posframe]] - The foundation library by tumashu
- [[https://github.com/tumashu/ivy-posframe][ivy-posframe]] - Posframe display for Ivy
- [[https://github.com/tumashu/helm-posframe][helm-posframe]] - Posframe display for Helm
- [[https://github.com/tumashu/company-posframe][company-posframe]] - Posframe display for Company

* Installation

** Using el-get

#+begin_src emacs-lisp
(leaf skk
  :ensure ddskk
  :custom ((default-input-method . "japanese-skk"))
  :config
  (leaf ddskk-posframe
    :el-get (ddskk-posframe :url "https://github.com/conao3/ddskk-posframe.el")
    :global-minor-mode ddskk-posframe-mode))
#+end_src

** Manual Installation

1. Clone or download this repository
2. Add the directory to your ~load-path~
3. Add the following to your configuration:

#+begin_src emacs-lisp
(leaf skk
  :ensure ddskk
  :custom ((default-input-method . "japanese-skk"))
  :config
  (leaf ddskk-posframe
    :load-path "path/to/ddskk-posframe.el"
    :global-minor-mode ddskk-posframe-mode))
#+end_src

* Usage

Enable ~ddskk-posframe-mode~ to activate the posframe tooltip:

#+begin_src emacs-lisp
(ddskk-posframe-mode 1)
#+end_src

Disable the mode to revert to the default tooltip behavior.

* Customization

Configure the tooltip position by setting ~ddskk-posframe-style~:

#+begin_src emacs-lisp
(setq ddskk-posframe-style 'point)
#+end_src

Available styles:
- ~point~ - Display at cursor position
- ~window-center~ - Center of the current window
- ~frame-center~ - Center of the frame
- ~window-bottom-left~ - Bottom left of the window
- ~frame-bottom-left~ - Bottom left of the frame
